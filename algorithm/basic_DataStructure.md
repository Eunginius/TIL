# 1. 배열이란?
같은 자료형의 변수로 이루어진 요소가 모여 직선으로 줄지어 있는 자료구조
  
  
### 선언
자료형 배열_이름[요소_개수];   
### 초기화
선언 = {초기화 할 값들을 comma(,)로 구분}   
중괄호 안의 값들이 0부터 (요소_개수 - 1) 인덱스까지 순서대로 배열에 저장된다.   
### 메모리 할당과 동적 객체 생성
#### calloc 함수 
헤더: #include <stdlib.h>   
형식: void *calloc(size_t nmemb, size_t size);   
해설: 크기가 size인 자료가 nmemb개만큼 들어갈 메모리를 할당한다. 할당 후 모든 비트가 0으로 초기화횐다.   
반환: 메모리 할당에 성공 -> 할당 영역 첫 번째 포인트, 실패 -> NULL 포인트
