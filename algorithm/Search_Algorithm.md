# 선형 검색
### 선형 검색 다루기   
**Linear Search or Sequential Search**   
: 요소가 직선으로 늘어선 배열에서 원하는 키값 찾을 때까지 맨 앞부터 순서대로 요소를 검색하는 알고리즘    

#### while문으로 구현
종료될 때까지 증가되는 cnt 변수를 이용해 배열에 접근하여 키 값을 찾는다.   
종료 조건   
1. 검색할 값을 발견하지 못하고 배열 끝을 지나간 경우(검색 실패 -> -1 반환)   
2. 검색할 값과 같은 요소를 발견한 경우(검색 성공 -> 키값 반환)   

#### For문으로 구현   
배열 개수 n번만큼 반복문을 실행하여 키 값을 찾는다.    
도중에 키 값을 찾았다면 키 값을 반환하며 종료한다.   

### 보초법 다루기
원래 배열 요소 맨 끝에 검색하고자 하는 키값과 같은 값을 저장한다.   
이를 검색하기 전에 준비하는 **보초(sentinel)** 라고 한다.   
#### 장점   
원하는 키값을 찾기 못했을 때 필요한 종료 조건이 없어도 된다.   
즉, 종료 판단에 대한 비용을 2회에서 1회로 반이나 줄이는 방식이다.    
#### 구조   
1. 검색값 key를 보초로 a[n]에 대입   
2. 배열 요소 순차 검색   
3. **return n?-1:i** 코드를 이용한다.   
- 찾은 값이 보초이면 검색 실패를 알리는 -1 반환   
- 찾은 값이 n이 아닌 원래 데이터이면 값을 반환
